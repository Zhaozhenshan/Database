一、环境配置说明

	1 Mysql 版本 5.7.24
	2 PHP   版本 7.2.14
	3 Apache版本 2.4.37
	4 我使用的是Wampserver自带的Php以及mysql,如需运行需要安装Wampserver
	5 试用说明：
	  把源码文件夹(改名为PHP后)放入wampserver文件夹下的www文件夹下
	  打开浏览器输入 localhost/PHP
	  点击login.html即可打开登陆界面
	  默认用户名为admin  密码为admin
